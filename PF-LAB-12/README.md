Objectives of Lab 12
Understand the concept of dynamic memory allocation in C.
Learn how to use malloc(), calloc(), realloc(), and free().
Allocate memory at runtime based on user input.
Differentiate between malloc (uninitialized) and calloc (zero-initialized) memory.
Learn to safely resize memory using realloc().
Avoid common issues like memory leaks and dangling pointers.
Practice handling arrays using pointers and dynamic memory.
