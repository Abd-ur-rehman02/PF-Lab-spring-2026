Objectives of Lab 09
To understand how to create and use functions in C
To learn how to pass values to functions and get results using return values
To practice breaking a program into small, manageable parts using functions
To understand the use of void and value-returning functions
To learn how to work with arrays in functions
To calculate values like sum, average, and maximum using functions
To understand the concept of pointers
To learn how to use pointers to access and modify data
To practice pointer arithmetic instead of array indexing
To understand how void pointers can store different data types
To learn how to use type casting with pointers
To work with 2D arrays using pointers
To understand how memory is accessed using pointers in multi-dimensional arrays
To learn about function pointers and how they help in calling functions dynamically
To build real-world logic using concepts of functions and pointers together
