Objectives of Lab 10
To understand and use string library functions like strlen(), strcmp(), strcpy(), etc.
To perform string validation, comparison, and manipulation in C programs.
To learn working with 2D character arrays (array of strings).
To store and process multiple strings using loops and arrays.
To implement searching and counting operations on strings.
To understand the concept of file handling in C.
To learn how to create, read, write, and append files using fopen().
To use functions like fprintf(), fscanf(), and fgets() for file operations.
To handle file errors (NULL pointer checking) safely.
To apply all concepts in real-world scenarios like records, reports, and logs.
